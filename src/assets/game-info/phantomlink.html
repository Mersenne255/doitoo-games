Cognitive Benefits

  • Memory Updating — Trains your ability to revise stored information when it changes
  • Pattern Recognition — Improves detection of relationships and connections between items
  • Adaptive Recall — Builds the skill of retrieving the most current version of memorized content
  • Attentional Monitoring — Sharpens your awareness of when and where changes occur
  • Binding and Unbinding — Strengthens the ability to form and dissolve associations between items in memory
  • Interference Resistance — Reduces confusion between old and updated information
  • Cognitive Flexibility — Encourages rapid mental adjustment as connections shift over time

How It Works

You are shown a set of items with connections between them. Some connections then change, and you must track which links are current. After the update phase, you are tested on the final state of the connections. Difficulty increases as more items and changes are introduced.

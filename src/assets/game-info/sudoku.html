Cognitive Benefits

  • Logical Deduction — Trains your ability to eliminate possibilities through systematic reasoning
  • Constraint Satisfaction — Strengthens the capacity to hold and manage multiple constraints simultaneously
  • Working Memory — Improves how many partial solution states you can maintain in active memory
  • Pattern Recognition — Develops the ability to spot naked singles, hidden singles, and pair patterns
  • Attention to Detail — Sharpens observation of which digits are missing in rows, columns, and boxes
  • Strategic Planning — Builds the habit of scanning for the easiest deduction before attempting harder ones

How It Works

Fill every row, column, and box with each digit exactly once. The game supports three grid sizes: 4×4 (digits 1–4), 6×6 (digits 1–6), and 9×9 (digits 1–9). Every puzzle has exactly one solution — no guessing required at lower difficulties.

Controls

Tap/Click: Select a cell, then tap a digit on the number pad to place it. Tap the same digit again to clear it.

Keyboard: Arrow keys to navigate, number keys to enter digits, P to toggle pencil mode, Backspace to erase, H for a hint.

Pencil Marks

Toggle pencil mode to annotate cells with candidate digits. When you place a final digit, pencil marks in the same row, column, and box are automatically cleaned up.

Hints & Give Up

Stuck? Request a hint to see the next logical step with an explanation of the technique used. If you give up entirely, the game walks you through the complete solution step by step — showing which technique solves each cell.

Solving Techniques

Naked Single — A cell has only one possible digit after elimination.

Hidden Single — A digit can only go in one cell within a row, column, or box.

Naked Pair — Two cells in a unit share the same two candidates, eliminating those digits from other cells.

Pointing Pair — A candidate in a box is restricted to one row or column.

Box Line Reduction — A candidate in a row or column is restricted to one box.

Difficulty

Difficulty 1–100 controls the grid size, number of given cells, and which techniques are needed. Lower levels use smaller grids and simple techniques. Higher levels use 9×9 grids with fewer givens and advanced elimination patterns.

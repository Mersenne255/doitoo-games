Cognitive Benefits

  • Logical Deduction — Trains your ability to eliminate possibilities through systematic reasoning
  • Constraint Satisfaction — Strengthens the capacity to hold and manage multiple constraints simultaneously
  • Working Memory — Improves how many partial solution states you can maintain in active memory
  • Pattern Recognition — Develops the ability to spot singles, pairs, triples, X-Wings, and other patterns
  • Attention to Detail — Sharpens observation of which digits are missing in rows, columns, and boxes
  • Strategic Planning — Builds the habit of scanning for the easiest deduction before attempting harder ones

How It Works

Fill every row, column, and box with each digit exactly once. Three grid sizes are available: 4×4 (digits 1–4), 6×6 (digits 1–6), and 9×9 (digits 1–9). Every puzzle has exactly one solution.

Controls

Tap/Click: Select a cell, then tap a digit on the number pad to toggle it. Tap the same digit again to remove it.

Multiple digits: You can enter several digits in one cell — they'll shrink to fit. When only one digit remains, it becomes your answer. Cells with multiple digits show an orange border.

Number pad: Buttons light up to show which digits are currently in the selected cell.

Keyboard: Arrow keys to navigate, number keys to toggle digits, Backspace to erase, H for a hint.

Tools

↩ Undo / ↪ Redo — Step back and forward through your moves.

💡 Hint — First tap highlights the next solvable cell. Second tap fills it in.

🏳️ Give Up — Opens a menu with three choices: Keep Trying to go back to the puzzle, Go home to save your progress and return to the game list, or Show Solution to see the step-by-step walkthrough.

Solution Walkthrough

When you choose Show Solution, the game walks you through the solution one step at a time. Each step highlights three types of cells:

Green — The cell being solved (the answer).

Amber — The cells that explain why the deduction works (the reason).

Red — Cells where candidates are eliminated (for advanced techniques).

Each step includes a detailed explanation with bullet points showing the reasoning. Advanced techniques (X-Wing, XY-Wing) are marked with a red badge. You can leave the walkthrough at any time.

Solving Techniques

Naked Single — A cell has only one possible digit after elimination.

Hidden Single — A digit can only go in one cell within a row, column, or box.

Naked Pair — Two cells in a unit share the same two candidates, eliminating those digits from other cells.

Hidden Pair — Two digits that can only go in the same two cells within a unit.

Naked Triple — Three cells in a unit whose combined candidates are exactly three digits.

Pointing Pair — A candidate in a box is restricted to one row or column.

Box Line Reduction — A candidate in a row or column is restricted to one box.

X-Wing (Advanced) — A digit appears in exactly two cells in two rows, sharing the same columns. Eliminates that digit from the rest of those columns.

XY-Wing (Advanced) — A pivot cell with two candidates connects to two wing cells, allowing elimination of a shared digit from cells that see both wings.

Backtracking

On rare occasions, especially at very high difficulty, no standard technique can make progress. When this happens, the solver tests a candidate digit in a cell and checks if it leads to a valid solution. In the walkthrough, these steps are marked with a red Advanced badge. This is not random guessing — it's systematic trial-and-error, the same approach used by competitive Sudoku solvers when logic alone isn't enough.

Difficulty

Choose your grid size (4×4, 6×6, or 9×9) and difficulty (1–100) separately. Difficulty controls the number of given cells and which solving techniques are needed. Lower difficulty means more givens and simpler techniques. Higher difficulty means fewer givens and may require advanced techniques like X-Wing and XY-Wing.

Game State

Your progress is saved automatically. If you leave and come back, you'll pick up right where you left off.
